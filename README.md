# zyan-skills

ZyanZealous 的 Claude Code 插件市场。

## 安装

```
/plugin marketplace add ZyanZealous/claude-skills
/plugin install ambition@zyan-skills
```

## 插件列表

### ambition

大规模软件开发全生命周期管理 skill，六阶段：

1. **DISCOVER** — 需求采集（分类问题库 + assumption log）
2. **DEFINE** — 需求定义（12 节 PRD 模板、user stories、MoSCoW、Given/When/Then AC）
3. **PROTOTYPE** — 原型创建（text wireframe / React artifact / API contract / ERD / sequence diagram）
4. **BREAKDOWN** — 需求拆分（Epic→Story→Task→Subtask，T恤尺码估点）
5. **TRACK** — 状态管理（任务状态机 + sprint burndown）
6. **REPORT** — 进度汇报（standup / sprint review / executive RAG status）

两种模式：自动规划（描述 + 深度，自动补足中间步骤）和自定义（用户定义流程，AI 补充细节）。

用法：`/ambition 我要做一个在线教育平台，标准深度`
